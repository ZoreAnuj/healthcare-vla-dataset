# Healthcare VLA Dataset

This project curates a specialized Vision-Language-Action (VLA) dataset for healthcare robotics, derived from the Open-H-Embodiment initiative. It aims to provide a foundational, open-source resource for training and evaluating generalist models that can understand instructions and perform physical tasks in medical environments.

## Key Features
*   Curated subset focused on healthcare-related robotic tasks and scenes.
*   Paired visual observations, natural language instructions, and corresponding action trajectories.
*   Processed and structured for ease of use in machine learning pipelines.
*   Serves as a benchmark for developing assistive robotics applications.

## Tech Stack
*   Python
*   Roboflow
*   PyTorch / TensorFlow (for model training with the dataset)

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/healthcare-vla-dataset.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Explore the dataset structure in the `data/` directory.
